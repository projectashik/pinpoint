## Description (paste into the Description box)
PinPoint Editor turns any web page into a live canvas. Open the toolbar, click an element, and change it on the spot — no DevTools, no code. Best of all, your edits are saved per page and come back after you reload, so you can mock up changes that actually stick.

WHAT YOU CAN DO
• Inspect — hover any element to see its selector, font-family, font-size, weight, line-height, color, and dimensions.
• Edit text — click an element and type. Great for trying new copy, headlines, and button labels.
• Move — drag any element anywhere on the page (applied as a non-destructive transform).
• Resize fonts — select an element and press the Up/Down arrow keys to change its font size (hold Shift for ±10px steps).

EDITS THAT PERSIST
Every change is saved locally for the current page and re-applied automatically when you reload. Your work isn't lost when the tab refreshes.

ENABLE / DISABLE TO COMPARE
Flip a single switch to temporarily turn all your edits off and see the original page — then turn them back on. Nothing is deleted. When you're done, "Clear" removes the saved edits for that page.

KEYBOARD & TOOLS
• Alt+Shift+V — show/hide the editor toolbar
• ↑ / ↓ — change font size (Shift = ±10px)
• Enter / Esc — commit a text edit
• Esc — deselect

WORKS EVERYWHERE
Built on Manifest V3 and runs on Chrome, Edge, and Firefox.

PRIVACY
PinPoint Editor collects nothing. There are no analytics and no network requests. All edits are stored locally in your browser (chrome.storage.local) and never leave your device.
